# 🧮 Goubroh Advanced Calculator

A modern and user-friendly advanced calculator built with Python. This application performs both basic and advanced mathematical operations with a clean and intuitive interface.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖ Multiplication
- ➗ Division
- 🧮 Modulus
- 📈 Exponentiation
- √ Square Root
- 📊 Logarithmic Functions
- 📐 Trigonometric Functions
- 🧠 Error Handling
- 🎨 Clean and Responsive User Interface

## 🛠️ Technologies Used

- Python 3
- Tkinter / CustomTkinter *(Update this based on your project)*
- Math Module

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/goubroh/Goubroh-Advanced-Calculator.git
```

2. Navigate to the project folder:

```bash
cd Goubroh-Advanced-Calculator
```

3. Install the required packages:

```bash
pip install -r requirements.txt
```

4. Run the application:

```bash
python calculator.py
```
